# xCombat

Projekt zaliczeniowy z programowania

## Opis

Gra 2D typu shooter wykonana w C++ oraz Qt/QML.
Sterujemy samolotem, strzelamy do przeciwników.

## Sterowanie

← → ↑ ↓ - ruch

SPACJA - strzał

ESC - pauza

## Funkcje

- 4 poziomy trudności
- 4 rodzaje przeciwnkików
- przeciwnicy różnych typów
- system punktów
- lista najlepszych wyników
- pasek życia
